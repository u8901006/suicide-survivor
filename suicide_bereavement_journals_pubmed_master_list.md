
# Suicide Bereavement / Survivors of Suicide Loss  
## English Markdown Master List: Journals, Keyword Bank, and PubMed Search Templates  
**Last updated:** 2026-04-17

## Scope
This is a **practical, PubMed-oriented master list** for research on **suicide bereavement**, **survivors of suicide loss**, and closely related topics such as **prolonged grief**, **stigma**, **postvention**, **family impact**, and **help-seeking**.

Because there is **no single journal category devoted only to suicide-loss survivors**, a good search strategy usually spans:

1. **Suicide-specific journals**
2. **Bereavement / grief / loss journals**
3. **Psychiatry / clinical psychology journals**
4. **Public health / social science journals**

> **Important note:** the phrase **“suicide survivor”** is ambiguous in the literature. It can mean:
> - a person who **survived a suicide attempt**, or
> - a person **bereaved by another person’s suicide**.  
>
> For PubMed, it is usually better to search with explicit phrases such as:
> - **suicide bereavement**
> - **bereavement by suicide**
> - **survivors of suicide loss**
> - **suicide-loss survivors**
> - **people bereaved by suicide**

## Practical Search Tips
- Start with **suicide-specific journals** first for the highest yield.
- Then expand to **grief / bereavement journals**.
- Then widen to **psychiatry** and **public health / social science** journals.
- For more precise results, add one of these concept pairs:
  - **stigma + shame**
  - **prolonged grief + depression**
  - **postvention + support**
  - **family + adolescent**
  - **help-seeking + service use**
- If a search is too broad, remove **"suicide exposure"** and keep only explicit bereavement terms.
- If you need a **systematic review sweep**, search the same core block across all suicide-specific, bereavement, psychiatry, and public health journals.
